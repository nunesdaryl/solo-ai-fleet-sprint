1

The Channel Distiller

~2 hrs/week back

Replaced: Re-watching long videos to find one thing.

Does: Ingests an entire channel, pulls every transcript, distills it into one searchable SKILL.md. Ask it a question; it answers from the whole channel.

Pattern: transcript fetch → structured distillation → searchable knowledge file. Fallback chain when captions are blocked (API → browser grab → local whisper).

Build-your-own v1: Pick ONE channel you keep re-watching. List video IDs → fetch transcripts → write each to markdown with title + key takeaways. That's v1.

2

The Signal Scout

~1.5 hrs/week back

Replaced: Doomscrolling X 'to stay current.'

Does: Scrapes a target account daily, transcribes their videos, ranks every post P0–P3 by relevance to YOUR goals — a 5-line digest instead of a feed.

Pattern: scrape → LLM scores each item against your mission → daily ranked digest. The ranking prompt is the whole product.

Build-your-own v1: One handle, 15 latest posts, one LLM call tagging each 'act now / this week / log / ignore.' Read only the top tier.

3

The Discussion Synthesizer

~1 hr/week back

Replaced: Reading every thread in your paid community.

Does: Pulls posts + comments, deep-reads only threads with real debate (≥3 replies), writes a top-5 synthesis: the insight, the pushback, the resolution.

Pattern: scrape posts + comments → filter to substantive threads → per-thread deep-read → ranked synthesis. It refuses to invent discussions on a quiet day.

Build-your-own v1: Export your community feed, filter to high-comment posts, summarize each in 3 bullets (insight / objection / resolution).

4

The Draft Engine

~1 hr/week back

Replaced: The blank page.

Does: Turns each day's ranked signals into 2–3 thread skeletons staged for you to approve, edit, or kill — so you never start from zero.

Pattern: today's top signals → angle selection → outline (not full prose) → staged for human review. It drafts; you decide.

Build-your-own v1: Feed your top 3 signals to an LLM with 'give me 3 thread angles + a 6-tweet outline each.' Edit the best one.

5

The Night Shift

the one that makes the other 4 trustworthy

Replaced: Babysitting scripts, wondering if they ran.

Does: Schedules all the above, watches for SILENT failures (fires on the absence of activity, not just errors), records what shipped, pings you only when a human is truly needed.

Pattern: schedule + staleness watchdog + source-of-truth log + alert-only-on-real-failure. Most automation dies silently; this catches it.

Build-your-own v1: For every agent, add a second job that checks 'did agent X produce fresh output in the last 24h?' If not, alert. That habit is the difference between a demo and a system.
